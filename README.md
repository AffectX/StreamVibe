# StreamVibe 
Адаптивный frontend стримингового сервиса

## Описание

Разработал адаптивный frontend стримингового сервиса с компонентной структурой и большим количеством интерактивных UI-элементов. Реализовал динамическое поведение интерфейса с использованием ванильного JavaScript.

## Функциональность

- Многостраничный интерфейс (6 страниц)
- Адаптивная верстка для различных устройств
- Табы и аккордеоны для переключения контента
- Слайдеры для отображения коллекций контента
- Мобильное меню (бургер-меню) с управлением состоянием
- Кастомный видеоплеер
- Маски для полей ввода
- Кастомный select

## Что реализовано 

- Интерактивные компоненты на ванильном JavaScript
- Переиспользуемые UI-блоки и компонентный подход
- Работа с событиями DOM и динамическое изменение интерфейса
- Структурированная верстка по методологии БЭМ
- Чистая и масштабируемая структура стилей на SCSS

## Технологии

- JSX
- SCSS
- JavaScript
- Vite
- Minista
- БЭМ
